Suma de Números

Resultado

Crear un archivo HTML que contenga y realice lo siguiente: Debe tener una etiqueta h1 como título que diga “Suma de Números”, un pequeño formulario con dos input de texto, y un botón que tenga como valor sumar. Este botón debe permitir sumar dos números que digite el usuario y mostrar el resultado a través de una alerta. Además, cuando se cargue la página se debe mostrar un mensaje con un cuadro de diálogo preguntando si “Desea sumar números?”. Si se hace clic en la opción Aceptar debe mostrar un cuadro prompt donde pide digitar un número y luego el segundo número. Cuando se digiten los números en el cuadro prompt se debe mostrar un mensaje en una etiqueta p al final del formulario con el resultado de la suma de los números.
